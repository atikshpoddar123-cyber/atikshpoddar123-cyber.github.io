How To Play Canvas

Simple loop: capture, build, train soldiers, and push outward.
1. Select
Use WASD or arrow keys and press Enter.
2. Capture
Press E on nearby neutral/enemy land.
3. Build
Use 1 House, 2 Market, 3 Barracks, 4 Wall.
4. Train + Move
Q to train. Tab unit cycle, Enter to set unit target.
Tip: build a stable economy first, then pressure with soldiers.
Press H anytime to open/close this tutorial.

Keyboard Commands

All gameplay actions are available with keyboard input.

  • WASD / Arrows move cursor
  • Enter select tile / issue selected unit move target
  • E capture selected tile
  • 1 House, 2 Market, 3 Barracks, 4 Wall
  • Q train soldier, Tab cycle units, H help
